# Git Command

made by @LeoDuhz, 2020

only some basic common commands of git 



```bash
$ git init
$ git init [project-name]
$ git config [--global] user.name "[name]"
$ git config [--global] user.email "[email address]"
$ git config --list
$ git add [file]
$ git add [dir]
$ git add .
$ git rm [file]
$ git commit -m [message]
$ git commit [file1] [file2] ... -m [message]
$ git commit --amend -m [message]
$ git status
$ git log
$ git diff
$ git remote rm origin
$ git remote add origin git@github.com:LeoDuhz/Linux-command.git
$ git remote show
$ git checkout [file]
$ git checkout [commit] [file]
$ git checkout .
$ git reset --hard           # with workspace and index reset
$ git reset --keep [commit]  # with workspace and index not changed

```
